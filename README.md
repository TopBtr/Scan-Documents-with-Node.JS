Online Document Imaging and Uploading with Dynamic Web TWAIN & Node.js
=======================================================================

The sample shows how to upload files with Struts. 


How to Run:
-----------
1. `npm install formidable@latest`
2. `npm install express`
3. `node server.js`
4. visit `localhost:2014`
